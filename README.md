# 总体架构
## 用户对话记录获取
1. 登入对应AI Chat Web端账号
2. 保存登录信息以便后续使用
3. 获取对话记录
   * (没有分组的情况)--> 直接根据element selector获取记录item
   * 3.2 (有分组的情况)--> 需要点击展开分组 --> 根据element selector获取记录item
4. 依次点击对话记录item, 等待详细对话加载
5. 选中菜单按钮触发点击事件唤出选择框
6. 寻找选择框中的下载menuitem并光标悬浮或直接点击
7. 等待进一步弹出下载选项框
8. 点击下载
   * 直接进行下载
   * 弹出下载确认框
